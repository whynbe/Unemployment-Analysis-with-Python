# 📉 Task 2: Economic Shockwaves – Unemployment Analysis with Python

The Hook: Numbers tell a story, and the year 2020 had a lot to say.


The Description: This repository dives deep into the socioeconomic impact of the COVID-19 pandemic through the lens of unemployment data. Moving beyond simple metrics, this project utilizes Python to aggressively clean, explore, and visualize the percentage of unemployed individuals over time. By highlighting key seasonal patterns and the stark spikes of lockdown periods , this exploratory data analysis (EDA) transforms raw rows into actionable policy insights.
An exploratory data analysis project examining unemployment rate percentages using Python. The analysis involves comprehensive data cleaning, exploration, and visualization to identify key patterns and seasonal trends, with a specific focus on investigating the economic impact of Covid-19 on unemployment metrics.
